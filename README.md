# SquareTowers
[ТЗ тестового задания](https://github.com/diemonic1/Square-Towers/blob/main/ТЗ/ТЗ_Unity_Kids.pdf)

Примечания:
- на выполнение тестового задания было затрачено около 7 часов
- из папки _BUILDS можно скачать билды игры для андройд или ПК
- чтобы взять квадрат из скрола, нужно удерживать на нем палец илит курсор. Время удержания задается в игровом конфиге, как и цвета квадратов для отрисовки
- вместо 20 одинаковых картинок квадратов используется одна, которая перекрашивается согласно конфигу цветов
- сервис конфигов сделан с предположением, что в будущем конфиги будут скачиваться с сервера
- на некоторых разрешениях можно увидеть, как картинки на фоне склеиваются (картинки должны быть заменены на бесшовные текстуры)
- новые квадраты в башне смещаются относительно предыдущего квадрата. У высокой башни верхние квадраты могут очень сильно отклониться в сторону относительно нижних, и при удалении квадратов в середине верхние квадраты не будут стоять на нижних. Нужно либо реализовать разрушение башни в таком случае, либо изменить генерацию позиции, чтобы квадраты не могли сильно отклонится в сторону
